📘 Подсказки

  • querySelector / querySelectorAll — основной способ поиска элементов.
  • textContent vs innerHTML — для пользовательских данных всегда используйте textContent.
  • createElement + append — создавайте узлы через JS.
  • classList.toggle / add / remove — управляйте внешним видом через классы.
  • element.remove() — удаляет элемент из DOM напрямую.
